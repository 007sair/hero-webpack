# 开发相关

``` bash
# 开发环境
npm run dev

# 生产环境
npm run build
```

**开发环境：**

开发环境不压缩js、css，没有合并雪碧图，没有转换base64

**生产环境：**

压缩js、css，合并雪碧图、转换base64
